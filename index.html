A platform for learning and improving juggling skills

The designed platform enables the user monitoring your own progress. Among other things, it allows you to record the progress of learning through the number of catches objects during juggling, which can be a determinant of the progress in learning juggling. Using the implemented juggler assistant application, the number of catches is counted and saved, so the juggler does not have to write the results himself. The results are automatically written each time you practice juggling. Thanks to this the user can see how fast his progress in learning juggling grows and compare this result with another one a juggling person. The application was written in C ++ using the OpenCV library to process the recorded image using the camera in real time.
