A platform for learning and improving juggling skills

The designed platform enables the user monitoring your own progress. Among other things, it allows you to record the progress of learning through the number of catches objects during juggling, which can be a determinant of the progress in learning juggling. Using the implemented juggler assistant application, the number of catches is counted and saved, so the juggler does not have to write the results himself. The results are automatically written each time you practice juggling. Thanks to this the user can see how fast his progress in learning juggling grows and compare this result with another one a juggling person. The application was written in C ++ using the OpenCV library to process the recorded image using the camera in real time.

Mobile asistance

One of the main goals of the application is the reminder module about the necessity of taking medications at a defined time. When the user receives a reminder to accept the medicine, the application emits a special notification sound until the person takes the medicine and confirms it in the application by clicking the confirm button (he can also refuse to take the medicine at the time of which the guardian will be notified). At this moment, the application generates the appropriate SMS content, which is sent to the guardian that the medications have been accepted. An older person's guardian has at his disposal a module of settings in the application on the elderly person's phone in which he can define the time of taking the medicine (morning, noon, evening), phone number to which the message should come and battery level below which the elderly person is to be notified of the need to connect the phone to the charger. The second main purpose of the application is the ability to notify the carer by the elderly about their well-being and basic needs. The only thing an elderly person has to do is click the appropriate button, the application will generate the appropriate content and send it to the maintainer. The application has been designed to be easy and user-friendly (large letters and large buttons).
